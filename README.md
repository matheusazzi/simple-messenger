# Simple messenger

## Dependencies

 * Ruby 2.3.1
 * Rails 5.0.x
 * PostgreSQL 9.3
 * Redis

## Installation

*There's a setup file, so you can just use:*

```
git clone https://github.com/matheusazzi/simple-messenger.git
cd simple-messenger
./bin/setup
```
